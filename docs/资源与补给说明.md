# 潜空间矿工 · 场上资源与补给道具说明

游戏内积分在 UI 上显示为 **培养积分**（代码里与 `$` 分数同源）。下列 **基础学分** 为未叠加任何补给加成时的数值；加成在「补给道具」一节说明。

---

## 一、场上资源类型（共 8 种）

每种物体在关卡生成时，会从该类型的 **显示名称池** 里按关卡与序号轮换取一条，作为场上悬浮标签（同一类型不同关可能看到不同梗名）。

| 内部类型 | 体量 | 基础学分 | 拉回权重 | 游戏内可能显示的名称（标签池） |
|:---|:---:|:---:|:---:|:---|
| `DATA_POINT` 小数据 | 半径 15 | **50** | 1 | 先导课实验数据、表征空间初探、有效训练样本、课程小作业成果 |
| `DATASET` 数据集 | 半径 25 | **150** | 3 | 组会展示用集、跨模态标注数据、验证与消融集、高质量训练池 |
| `CORPUS` 大型语料 | 半径 40 | **500** | 8 | 学院联合语料、开源大模型数据池、多模态知识库、校企共建数据集 |
| `ROCK` 绊脚石 | 半径 30 | **10** | 10 | 盲审意见 #2、培养环节 DDL、机房排队、显存告急、环境配置玄学 |
| `DIAMOND` 成果钻石 | 半径 12 | **1000** | 1 | 顶会 Oral、SOTA 突破、学院表彰成果、核心算法创新 |
| `BAG` 神秘袋 | 半径 20 | 见下文 | 4 | 校企课题彩蛋、神秘算力券、黑盒基础模型、预研种子基金 |
| `TNT` 集群维护窗 | 半径 20 | **0**（不得分） | 1 | 集群维护窗口、GPU 过热保护、误触 rm -rf |
| `INTERN` 实习生 | 半径 18 | **20**（普通） / **1020**（捧钻） | 2（捧钻时略增） | 同届博士互助、低年级博士生递样本、跨组联培伙伴、学术交流搭子 |

### 捧钻实习生（`INTERN_D`）

- 关卡配置里写作 `INTERN_D`，生成到场上仍是 **实习生**模型，但带 **手捧钻石**。
- 学分在生成时写死为 **20 + 1000 = 1020**（再叠加「成果认定证书」等对 💎 的加成，见下节）。

### 神秘袋（`BAG`）拉回后

- **培养积分**：`Math.floor(随机[0,1) × (320 + 140×luck)) + (40 + 25×luck)`（`luck` 初始为 1，买「随机种子」每买一次 +1，本关内生效）。  
  - 例：`luck = 1` 时约为 **65～524** 之间的整数。
- **额外概率**（随 `luck` 提高，各有上限约 42%）  
  - 再得 **稿件回收次数 +1**  
  - **大力抓取**：本关 `strength +2`（拉回更省力）

### 维护窗（`TNT`）

- 钩到后爆炸，**约 120 像素半径**内物体被清掉，**不得分**；钩子进入回收。

## 二、补给加成如何加在学分上（本关生效）

以下数值在 **进入关卡时** 根据本阶段在补给站**已购道具**汇总（每种道具每阶段最多买 1 次），在拉回结算时叠加：

| 条件 | 叠加规则 |
|:---|:---|
| 抓到 `DIAMOND`，或抓到捧钻 `INTERN` | 基础分 + `diamondValueBonus`（**成果认定证书**每件 +200） |
| 抓到 `ROCK` | 基础分 + `rockValueBonus`（**代码调试特训** +55；**Debug 急救包**再 +40，可叠加） |
| 抓到 `DATA_POINT` | 基础分 + `dataPointBonus`（**数据采样增强** +40） |
| 抓到 `DATASET` | 基础分 + `datasetBonus`（**数据采样增强** +100） |
| 抓到 `CORPUS` | 基础分 + `corpusBonus`（**语料放大镜** +150） |

例：基础钻石 1000，买过 1 次成果认定证书 → 拉回得 **1200** 分（若再买第二次证书需下一阶段进店，同一阶段不可重复购买同种道具）。

## 三、补给站道具（共 11 种）

每进入一次补给站，从 11 种里 **随机上架 3～6 种**，价格在表内 **最低价～最高价** 间随机。下列 **作用** 均指 **买下后、后续关卡中生效**（与货架上简短文案一致，此处写全机制）。

| 类型 | 货架名称 | 价格区间（约） | 作用（机制） |
|:---|:---|:---:|:---|
| `COFFEE` | 学术交流区续命拿铁 | $35–320 | 本关 **收回钩子速度**：`pullSpeedMultiplier +0.5`（与 `BASE_PULL_SPEED` 相乘，拉回更快）。 |
| `GPU` | 学院算力配额（H100） | $120–720 | 本关 **拉力**：`strength +3`（公式里减轻 `weight/strength`，重物更省力）。 |
| `SHREDDER` | 稿件回收通道 | $28–260 | 本关获得 **稿件回收次数 +1**（可叠加多次进店购买，每阶段每种限 1 次）；按 ↑ 或点 UI 消耗 1 次可丢弃当前钩上物等（玩法内操作）。 |
| `SEED` | 学院标准随机种子 42 | $70–480 | 本关 **luck +1**：神秘袋**学分区间上移**，且袋子里 **+回收次数 / +大力抓取** 的概率提高（有上限）。 |
| `DIAMOND_CERT` | 成果认定证书 | $95–620 | 本关拉回 **钻石**或**捧钻实习生**时，培养积分 **+200**（每件证书 +200，可与其他加成相加）。 |
| `ROCK_HANDLER` | 代码调试特训 | $55–380 | 本关拉回 **绊脚石**时，培养积分 **+55**。 |
| `DEBUG_KIT` | Debug 急救包 | $65–420 | 本关绊脚石 **再 +40** 分；**钩子碰撞半径 +5**（与基础 `HOOK_RADIUS` 叠加）。 |
| `WIDE_HOOK` | 宽口径探索钩 | $45–340 | 本关 **钩子碰撞半径 +12**。 |
| `TIME_BONUS` | 培养时限延期条 | $88–520 | 本关倒计时 **+18 秒**（在关卡基础秒数上叠加，例如 42→60）。 |
| `SAMPLER_PRO` | 数据采样增强 | $80–480 | 本关 **小数据 +40**、**数据集 +100** 分（拉回时叠加在对应类型基础分上）。 |
| `CORPUS_LENS` | 语料放大镜 | $100–560 | 本关 **大型语料 +150** 分。 |

### 汇总

- **场上可抓取资源类型**：**8** 种（`ItemType`）。  
- **补给道具种类**：**11** 种（`SHOP_ENTRY_DEFS` 条目数）。
