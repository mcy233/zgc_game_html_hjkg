UI 布局拖动工具

像 Unity Canvas 一样拖动/缩放 UI 模块。用于快速校准 9:20 游戏界面、补给站、弹窗等布局，最后导出 JSON 给开发接入。

场景

添加模块

切图素材联动

如果切图工具导出的 PNG 覆盖保存到旧路径，刷新本工具即可看到新图。浏览器不能直接把文件写入项目目录，所以临时导入只在当前页面会话有效。

快捷操作

方向键微调，Shift + 方向键移动 10px。拖动右下角橙点可缩放。

图层

720×1600
