## 1. Figma 与 Cursor MCP（重要说明）

### 1.1 为什么「团队主页」链接没法用？

形如  
`https://www.figma.com/files/team/…/recents-and-sharing`  
的地址是 **文件列表页**，没有 `figma.com/design/{fileKey}/…` 结构，MCP **无法**从中解析 `fileKey` / `node-id`，也 **不能**替你「扫团队里有哪些文件」或自动点开某一帧。

**你需要做的**：打开 **具体设计文件** 后，复制带 **`figma.com/design/`** 的链接；若要指定某一画板，用 **Copy link**（带 `node-id=`）。

### 1.2 已为你创建的空设计文件（可在此上继续画）

已通过 MCP 在你的团队下新建（若看不到，在 Figma 左侧 **Drafts / 团队项目** 里找同名文件）：

- **打开设计稿**：[潜空间矿工 · 中关村AI博士学院 UI](https://www.figma.com/design/4ZEz5H8zZWSOKtp66JJsE0)

我本想用 MCP 在该文件里 **自动生成线框画板**，但当前账号 **Figma MCP Starter 调用次数已达上限**，`use_figma` 未能执行完。你可以：

- **方案 A**：升级 Figma 计划以恢复 MCP 写入额度，再让我执行一次画板脚本；或  
- **方案 B**：你本人在该文件里新建 Frame，按游戏界面自行排版，把 **带 node-id 的链接** 发我，我用 `get_design_context` 读（该工具若也受限，以你本地导出 PNG 为准）。

### 1.3 日常协作你要做的

1. 打开上面的设计文件（或你自己的 `figma.com/design/xxx/...`）。  
2. 选中 Frame → **Copy link**（含 `node-id=`）。  
3. 把链接发到对话里，用于对齐前端。

## 3. 雪碧图（一张 PNG 多图标）规范

**目的**：少跑几次生图、统一光影，代码里用 `drawImage` 切格子。

- **画布**：例如 **512×512** 或 **1024×256**；背景 **完全透明**。  
- **排列**：**等分网格**，例如 4×4（16 格），每格 **留 4～8px 内边距**，图标不贴边。  
- **风格**：**统一线宽、统一光源**（建议左上光）；**扁平或微渐变 2.5D**，不要照片写实。  
- **交付时请附带**：`cols`×`rows`、总像素宽高、**每一格顺序**（从左到右、从上到下列表）。  
- **命名示例**：`sprites/items-game-4x4.png` 放进 `public/assets/`。

**顺序建议（与游戏一致）**：  
1 小数据点 → 2 数据集文件夹 → 3 大语料 → 4 绊脚石石 → 5 成果钻石 → 6 神秘袋 → 7 TNT → 8 实习生；空位可留透明。

## 4. AI 生图提示词（中文 · 风格统一）

**用法**：下面 **【风格母版】** 同一段复制到 **每一次** 生图提示的 **最前面**（或最后面，按你用的工具习惯），再拼接对应 **【任务句】**。这样全套素材会统一为：**扁平矢量、2～3px 深色描边、左上光源、轻微渐变、Q 版但不幼齿、中关村 AI 博士玩梗、不要照片写实、不要血腥恐怖**。

### 【风格母版】—— 每张图都带上

```
【风格统一】扁平矢量游戏插画，2～3 像素深色描边，光源统一在左上方，轻微立体渐变（不要强烈 3D 渲染），
配色以深蓝灰背景层次 + 琥珀金点缀 + 天青色科技高光为主，幽默玩梗、适合大学生与研究生，不低俗不暴力，
北京中关村学院 AI 博士生培养主题，禁止照片级写实，禁止杂乱水印与多余文字。
```

### 4.1 整张道具雪碧图（一张 PNG 多格，推荐）

**【任务句】**（接在母版后面）：

```
【画面】透明背景，总尺寸 1024×1024 像素，严格 4×4 等分十六宫格，每格内容居中，格与格之间留均匀空隙，
每格内图标约占格子的 70%，风格与描边与母版一致。

【从左到右、从上到下的格子顺序】
第 1 格：发光小芯片，象征「先导实验小数据」；
第 2 格：绿色文件夹，角上有小型神经网络节点图案，象征「数据集」；
第 3 格：书本叠放 + 简化服务器机柜剪影，象征「大语料库」；
第 4 格：灰色裂石，石上有一个卡通小 bug 图标，象征「审稿意见 / 绊脚石」；
第 5 格：青色多面钻石带细小星芒，象征「顶会级成果」；
第 6 格：神秘钱袋上面有问号，象征「预研种子基金彩蛋」；
第 7 格：红色卡通炸药桶，桶身可有一条极小的虚构标签写「维护窗口」（字要清晰可辨但占画面很小）；
第 8 格：简笔 Q 版小人抱笔记本电脑跑步，象征「本科生科研助理 / 实习生」；
其余空格保持透明，不要填充杂点。
```

若模型画不出规整网格，可改为 **分 8 次单图标生成**（每次仍带【风格母版】），再在 Figma / PS 里拼成雪碧图。

### 4.2 矿层背景（单张，无 UI）

**【任务句】**：

```
【画面】横版 800×600，游戏「潜空间矿层」背景，深蓝到近黑的纵向层次，隐约水平岩层纹理，
极淡的天青色电路线条与少量柔光粒子，像神经网络在地下延伸，无角色、无按钮、无文字，
对比度偏低以便前景道具突出，整体安静、科研感、略带幽默的科幻氛围。
```

### 4.3 街机外框 + 中间挖空（给游戏画布镶边，可选）

**【任务句】**：

```
【画面】横版约 1600×900，复古街机塑料外框，琥珀黄面板 + 深褐描边，正面平视，
中央留出规整矩形透明或纯深色区域以嵌入 800×600 游戏画面，四角可有轻微螺丝装饰，
顶栏位置可暗示积分条区域但不要写具体数字与文字，矢量扁平，与母版配色一致。
```

### 4.4 补给站木纹条（可平铺）

**【任务句】**：

```
【画面】横条约 1024×256，浅色橡木卡通木纹，暖琥珀色调，横向木板拼接，结节尽量少，
可无缝平铺用于游戏「学院补给站」货架背景，无文字，扁平矢量，与母版描边风格协调。
```

### 4.5 启动页背景插画（可选，无 UI 字）

**【任务句】**：

```
【画面】竖版 9:16，深夜学院楼剪影 + 中关村的科技感天际线简化线条，前景一个戴博士帽的卡通小人剪影望向「发光的潜空间入口」，
整体励志又轻松，无真实校徽与校名书法，无具体人脸照片，与母版配色与描边一致。
```

### 4.6 若中文模型效果不稳定（备用英文一句）

可在母版后追加一句：`flat vector, mobile game asset, consistent outline width, no photorealism` —— **仍以中文母版为主**，英文仅作辅助。
